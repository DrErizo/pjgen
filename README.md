# pjgen
Project generator for multiple languages written in c99.

# Supported languages
* C  
* C++  
* HTML/CSS  
* Python
* More coming soon...  

# Usage
pjgen \<lang\> \<proj name\> \<optional flags\>  
type ```pjgen --help``` for more help

# Install
* ```git clone http://github.com/rotth/pjgen.git```  
* ```cd pjgen```  
* ```make install```
