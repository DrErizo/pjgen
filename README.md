# pjgen
Project generator for multiple languages written in c99.

# Supported languages
* C  
* C++  
* HTML/CSS  
* More coming soon...  

# Usage
* ```pjgen c "chad-music-player"```   
* ```pjgen cpp "i-like-bread"```   
* ```pjgen web "epic-facebook-clone"```  

# Install
* ```git clone -b stable http://github.com/rotth/pjgen.git```  
* ```cd pjgen```  
* ```make install```
